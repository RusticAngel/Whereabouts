# FindMe — Demo Video Script (45 seconds)

## Setup
- Open `https://whereabouts-navy.vercel.app/demo` in a browser or on your phone
- Clear session storage so tutorial coach marks appear:
  - Chrome DevTools → Application → Session Storage → clear
  - Phone: close tab, reopen
- Start screen recording

## Recording Tips

- **Phone**: iOS Screen Recording (Control Center) — best shows the mobile experience
- **Desktop**: OBS Studio (free) — crop to a phone-sized window for authenticity
- **Audio**: Speak clearly, or just record the screen and add narration in post
- **One take**: Practice the flow once before recording. The whole loop is under a minute
- **No login needed**: Demo route is auth-free — just tap and go
